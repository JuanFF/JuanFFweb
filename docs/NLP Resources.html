Disease-treatment NER [April 2020]

This is a Named Entity Recognition algorithm to extract named diseases and treatments from sentence-like text.



For the sentence
Favipiravir to treat COVID-19

NER will return the following
{ "treatment": "Favipiravir", "disease": "COVID-19" }

Both entities must be semantically related to be well labeled. This entity-treats-entity relationship must be explicit in text; otherwise, NER will return an <empty> tag.

Consider the following
Favipiravir can't treat COVID-19

The result will then be
{ "treatment": "<empty>", "disease": "COVID-19" }

The algorithm precision score is 0.89. Performance varies according to text source or style. I used Twitter (healthcare-related tweets) and PubMed (research paper titles) as text sources for training and testing.

Healthcare Twitter User Classifier [April 2020]

This is a text classifier that categorizes user names and descriptions from Twitter accounts related to health and medicine.

As Twitter users provide information on themselves in their user names and descriptions, the classifier reads from this data (name, description) to return a descriptive label for each profile. This is the current classifier's tag set:

  • Academia
  • Publishing source
  • Doctor
  • Professional
  • Medical business
  • Interested in healthcare
  • News source
  • Healthcare initiative
  • Association
  • Generic

In all cases, the classifier must receive the Twitter user name and description as input data. If the input profile doesn't belong to any of these categories, it returns <empty>.

User profile classification may be useful when managing loads of health information on Twitter. The classifier's precision score is 0.86

Corpus disease-treatment [April 2020]

Find here an annotated corpus of healthcare headlines. It consists of 314620 sentence-like texts that mention one disease and its treatment. For each text, the JSON file encodes this information:

{ "message": "Botox as a treatment for bruxism",
  "treatment": "Botox",
  "disease": "bruxism",
  "source": "twitter",
  "date": "2018-04-22"  }


treatment and disease are named entities that were automatically labeled using the disease-treatment NER. The text from message comes from healthcare users' tweets, PubMed paper titles, and titles from studies in ClinicalTrials.gov (as specified in source).

This corpus has information from 2018 and 2019. I have been working on it for training and testing NLP algorithms.

All texts are publicly available on Twitter, PubMed and Clinical Trials. This corpus data is not intended for medical purposes, but for training or evaluating machine learning models.

Corpus of health twitter accounts [April 2020]

The following corpus consists of 104698 short descriptions from health Twitter accounts. Each text is a short bio or description where the account owner introduces himself. 

  { "description": "Scientist, cardiologist, geneticist",
    "profile": "Doctor" }
  { "description": "Cell Biologist at Medical Research Institute",
    "profile": "Academia" }
  { "description": "Promote genetic diagnostic solutions for cancer",
    "profile": "Med Business" }
  { "description": "Institute for Bioengineering of Catalonia",
    "profile": "Institution" }


Corpus texts have been automatically labeled by the Twitter user classifier, with a precision score of 0.86
